

BEM VINDO/A AO MENU DE AJUDA!

SIMULAÇÃO:
ANTES DE INICIADA A SIMULAÇÃO, O UTILIZADOR PODERÁ ESCOLHER A VELOCIDADE PRETENDIDA:
 -INSTANTÂNEA -> O UTILIZADOR É REDIRECIONADO PARA O MENU FINAL DE SIMULAÇÃO; 
 -RÁPIDA OU LENTA -> O UTILIZADOR PODERÁ ACEDER AO MENU PAUSA ATRAVÉS DA TECLA "ESPAÇO" A UMA DAS VELOCIDADES ESCOLHIDAS

    POSTERIORMENTE É POSSÍVEL ACEDER A: 
     RESULTADOS.
     ESTATÍSTICAS.
     PESQUISAR(CLIENTES, PRODUTOS, ETC).

MENU PAUSA:
QUANDO ABERTO O MENU PAUSA, AS OPÇÕES DISPONIVEIS SÃO:
 -CONTINUAR SIMULAÇÃO
 -CONCLUIR SIMULAÇÃO
 -FECHAR CAIXA
 -ABRIR CAIXA
 -RESUMO DE INFORMAÇÕES

DEFINIÇÕES:
PERMITE AO UTILIZADOR ALTERAR OS PARAMETROS A USAR NA SIMULAÇÃO.

IMPORTAÇÃO DE DADOS:
 OS DADOS SÃO CARREGADOS DOS FICHEIROS:
 -CLIENTES, FORMATO: [ID] [NOME]
 -FUNCIONÁRIOS, FORMATO: [NÚMERO] [NOME]
 -PRODUTOS, FORMATO: [CÓDIGO] [NOME] [PREÇO €] [TEMPO_COMPRA(SEGUNDOS)] [TEMPO_CAIXA (SEGUNDOS)]

MENU FINAL:
NO MENU FINAL É POSSIVEL:
 -CONSULTAR RESULTADOS
 -ACEDER ÀS LISTAGENS DAS CAIXAS, FUNCIONARIOS,..
 -EXPORTAR PARA FICHEIRO OS RESULTADOS DA SIMULAÇÃO PARA A PASTA ../export
